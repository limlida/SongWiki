# 釉的技术演进

## 概述
釉 = 陶瓷表面的一层薄玻璃。三大成分：氧化硅（釉体）+ 氧化铝（黏着剂）+ 助熔剂。历代釉的演进，本质是**助熔剂**的变迁。

## 三代釉

### 第一代：石灰釉（商周 → 1953 年，约 3000 年）
- 助熔剂：氧化钙（来自草木灰 + 石灰）
- 来源：草木灰天然含氧化钙约 10%，可单独制釉。宋代发现不够用，开始加生石灰
- 景德镇经典制法：狼鸡草 + 石灰反复烧 3 次 → 淘洗 → 人尿沤数月
- 釉果（制釉瓷石）10 份 + 釉灰 1 份 = 精细瓷
- 釉灰越多→烧成温度越低，但釉面发灰（低档灰可器）
- 优点：透明坚硬，优于埃及盐釉
- 缺点：烧成温度范围窄（1280-1320℃），升降温快的窑炉不适应

### 第二代：石灰碱釉（南宋龙泉窑首创）
- 助熔剂：氧化钙 + 氧化钾（长石）
- 降低氧化钙比例，加入氧化钾
- 核心优势：**黏度高**——可上很厚釉而不流淌
- 直接产物：粉青（偏蓝乳浊）、梅子青（晶莹绿色），釉比胎厚
- 未普及，仅有龙泉窑使用

### 第三代：长石釉（1953 年至今）
- 助熔剂：长石（钾长石、钠长石等）
- 代表：5311 釉（1953 年第 11 次试验成功）
- 触发原因：煤窑升降温快，石灰釉不适应
- 优点：黏度大不流淌、烧成范围宽（1100-1400℃）、适应电窑气窑
- 缺点：透明度差、易裂纹
- 电力球磨→颗粒均匀→镜面反射→"贼亮"；古法水碓→颗粒不均→漫反射→光泽柔和
- 地壳中 60% 成分是长石，为何到近现代才用？——纯长石莫氏硬度 6，人力水力无法粉碎，电力球磨机出现后才可行

## 釉灰制作（景德镇，宋末→民国）
1. 地上铺狼鸡草，撒石灰，点火烧 6 小时
2. 灰烬上再铺草和灰，反复 3 次
3. 淘洗后入灰池，**人尿沤数月**
4. 沤过的釉灰颗粒细腻、不腐蚀坯体——唐以前没这道工序，釉面粗糙易脱落
