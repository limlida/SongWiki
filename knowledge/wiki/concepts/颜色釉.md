# 颜色釉

## 概述
在透明基础釉（白釉）中加入金属氧化物着色剂烧成的彩色瓷器。唐代砖瓦窑瓷工随手在釉水中掺进黏土→无意间打开大门："门外的铜红、铁黄、锰紫、钴蓝，扑面而来。"青白之争在元代结束，真正的胜利者之一是颜色釉（另一是彩绘瓷）。^[陶瓷简史:L511]

## 核心内容

### 金属着色剂色谱
| 金属 | 呈色 | 代表品种 |
|------|------|---------|
| 铁（Fe）低浓度+还原焰 | 青/绿/蓝 | 青瓷全系 |
| 铁（Fe）中浓度+氧化焰 | 黄/褐 | 酱色釉 |
| 铁（Fe）高浓度（8%+）| 黑 | 黑釉 |
| 铁（Fe）+还原焰 | 红（极难）| 祭红/郎红 |
| 铜（Cu）+还原焰 | 红 | 钧红/郎窑红 |
| 铜（Cu）+氧化焰 | 绿 | 汉绿釉/三彩绿 |
| 钴（Co）| 蓝 | 蓝釉/青花 |
| 锰（Mn）| 紫 | — |

### 经典品种
- **酱色釉**（东汉）：第一种颜色釉，铁5%
- **黑釉**（东汉）：铁8%+，宋代斗茶盛行
- **结晶釉**（宋代）：铁10%+氧化焰→析晶（油滴/兔毫/玳瑁）
- **钧红**（宋代）：铜红+牛骨粉乳浊剂
- **乌金釉**（康熙）：铁+钴+锰→漆亮黑色
- **三阳开泰**（雍正）：乌金+郎红各三层，黑红间青绿过渡
