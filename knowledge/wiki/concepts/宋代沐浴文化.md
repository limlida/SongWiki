# 宋代沐浴文化

## 概述

宋代是中国沐浴文化发展的重要阶段——公共浴室自宋代始有（史学界一般公认），从上层官员到平民百姓都有良好的沐浴习惯。成寻在巡礼途中多次记录沐浴经历及其价格，为研究宋代沐浴的普及程度和经济门槛提供了精确数据。僧人沐浴则将日常清洁与宗教仪式融为一体。

## 核心内容

### 公共浴室：市民消费

北宋时以盈利为目的的公共浴室遍布街巷。仁宗时"其居在宋门外，俗谓之浴堂巷"——甚至有街巷以浴堂命名。南宋临安"此行在城中有浴所三千"（马可波罗记），有时可同时容纳百余人。浴堂提供搓背服务——苏轼泗州沐浴词"寄语揩背人，尽日劳君挥肘"可证。

成寻记录的沐浴价格：

| 场合 | 价格 |
|------|------|
| 杭州浴堂，8人 | 80文（约10文/人） |
| 次年二月，单人 | 100文 |
| 次年四月，南浴堂 | 100文+50文 |
| 次年四月，单人 | 30文 |
| 忻口寨金山驿（近水井） | 便宜（未记价） |
| 郑州丰宁驿（买柴烧水） | 买柴120文 |

程民生分析"北宋时期在洗澡堂沐浴一次10文左右"——成寻记录基本吻合。价格波动反映地区差异（驿站需买柴则贵，近水井则便宜）。

### 文人士大夫的沐浴风尚

士大夫是沐浴的强烈爱好者。苏轼在泗州写下两首《如梦令》谈沐浴——"寄语揩背人，轻手轻手，居士本来无垢"。王安石不好沐浴，好友吴充、韩维"相约每一两月即相率洗沐"，号"折洗"。尚书左丞蒲宗孟最为极端——有大洗面、小洗面、大濯足、小濯足、大澡浴、小澡浴之分，每日两洗面两濯足，间日一小浴、间日一大浴。

士大夫相约沐浴不仅关乎个人卫生，还带有社会交往功能。

### 僧人沐浴：世俗与神圣之间

佛寺中设浴院专供僧人沐浴——成寻多次在国清寺浴院、太平兴国寺浴院沐浴。"僧者，净也"——沐浴既是日常清洁，也是宗教仪式。释迦牟尼诞生时有龙喷洒香雨沐浴，后在尼连禅河沐浴后顿悟得道。《十诵律》载沐浴五种利：除垢、身清、除寒冷、除身中风、得安稳。

太平兴国寺浴院"僧俗数百人集浴，室内暗，互不见形"——僧俗同浴是普遍现象。《丛林校定清规》对沐浴礼仪有数十条规定：不能赤脚、需穿鞋子、不能说笑、前后有人时需遮蔽身体等。

### 浴佛节：四月初八

"诸寺设斋，以五色香水浴佛，共作龙华会。"——以都梁香（青）、郁金香（赤）、丘隆香（白）、附子香（黄）、安息香（黑）为五色香水，从佛顶部灌下。"十大禅院各有浴佛斋会，煎香药糖水相遗，名曰浴佛水。"观礼者求浴佛水饮漱——宋人认为浴佛水带有灵性。
