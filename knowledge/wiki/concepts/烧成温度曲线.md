# 烧成温度曲线

## 概述
一件瓷坯从入窑到成瓷，经历六个温度阶段。每个阶段发生不同的物理化学变化。理解这条曲线，就理解了"泥做火烧"的本质。

## 六阶段详解

### 第一阶段：室温 → 300℃（排水）
- 不发生化学变化，仅排除坯体内物理水分
- 水分蒸发时体积急剧增大→可能撑裂坯体
- **入窑前必须晒干**：陶坯含水 <2%，瓷坯 <1%，大件 <0.5%
- 中空瓷坯必须开小孔排气→否则此时会炸开

### 第二阶段：350℃ → 600℃（排碳）
- 坯中有机碳氧化为 CO₂
- 600℃ 后单质碳开始氧化——若钻戒入窑，此时已化为灰烬

### 第三阶段：480℃ → 600℃（结构水排除）
- 三种原料分子式均为 Al₂O₃·2SiO₂·2H₂O，每个大分子含两个化学结合水
- 结构水完全蒸发→**坯体爆炸隐患解除**
- 可开窑门放出水蒸气

### 第四阶段：600℃ → 1000℃（杂质分解）
- 碳酸盐和硫化物分解（如碳酸钙 → 氧化钙 + CO₂）
- 陶器至 1000℃ 已烧成

### 第五阶段：1000℃ → 1200℃+（莫来石形成）
- 1000℃：石英+长石开始熔融→形成**颗粒状莫来石**（瓷器的骨架）
- 1200℃+：石英+长石继续熔融，颗粒状莫来石也熔解
- 坯体约 60% 液态，开始缓慢流动→**填平杂质烧失留下的空隙**
- 莫来石二次形成→这次是**针状**，比颗粒状更结实坚硬
- **变形主要发生在这个阶段**——坯体不均衡或支撑不够→歪斜甚至倒塌

### 第六阶段：1300℃+ → 冷却（玻璃相）
- 超过 1300℃：主要作用于釉面，坯体不再变化
- 降温阶段：液态坯体继续形成针状莫来石 + 方石英
- 两者混和，晶莹剔透→**玻璃相**
- "至此，火之幻化终结了，土变为新的物质：瓷。"

## 关键数据

| 指标 | 数值 |
|------|------|
| 烧失（重量减少）| 3-8% |
| 体积收缩 | ~10% |
| 紫砂朱泥收缩 | 可达 23% |
| 陶器烧成温度 | ~1000℃ |
| 瓷器烧成温度 | 1100-1320℃ |
| 纯高岭土需温 | 1400℃（7501 毛瓷） |

## 莫来石
- 无色透明晶体，自然界少有
- 一般仅出现于硅酸盐煅烧过程
- 两次形成：颗粒状（1000℃+）→ 针状（1200℃+）
- 针状比颗粒状更结实→是瓷器的"骨架"
