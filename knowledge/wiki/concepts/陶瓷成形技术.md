# 陶瓷成形技术

## 概述
将泥料变成器形的全部工艺。核心矛盾：泥料柔软易塑形，但干燥和烧制时会收缩变形——成形技术要在"可塑性"和"精度"之间找平衡。

## 技术演进链

```
捏塑/泥条盘筑（新石器时代）
  → 慢轮（仰韶文化，前4000年）
    → 快轮拉坯（商周）
      → 利坯修整
        → 印坯（外模/范，秦代）
          → 雕镶（方形器）
            → 石膏注浆（1870年法国发明，民国传入）
```

## 手工成形技法

### 拉坯（快轮）
- 利用轮盘旋转的离心力成形
- 传统陶车：檀木轴+石座+瓷质顶碗+栗木转盘，惯性旋转 5-8 分钟，40-150 转/分钟
- 流程：摔泥（挤出气泡）→ 扒开中心 → 扯喇叭 → 塑形 → 裁坯
- 只能做圆形器皿
- 宋代所有圆形瓷器均以拉坯为基础

### 印坯（内模）
- 拉坯后扣在圆形陶内模上拍打，统一尺寸
- 景德镇称内模为"死人头"，此工序称"拍死人头"
- 宋代各窑口普遍使用

### 利坯
- 拉坯基础上的精修，在利坯车上以钢刀修整厚薄
- 关键技法：**蓄泥（撑口泥）**——口沿微厚起箍，防烧制变形
  - 好的蓄泥肉眼看不出来，须用手摸感受曲线
  - "增一分太肥，减一分变形"
- 利坯工收入仅次于把桩（烧窑总管），需多年经验
- 最后一刀最紧要，"轻则厚，重则漏"，如同剑客出鞘

### 印坯（外模/范）
- 模具在外（古称"范"），区别于内模（古称"模"）
- "模范"一词即源于此
- 始于秦代，以陶或木制成，分段成形后粘合
- 异形器（花瓣碗、方形器）必用此法
- 宋代定窑印花即用此技术：内模刻花纹→扣坯→花纹完整拓在器壁

### 雕镶
- 泥片擀压裁切后以泥浆粘合
- 用于方形器（镶器），比圆器贵
- 宜兴紫砂壶拍身筒即属此类

## 现代成形：石膏注浆法

- 1870 年法国发明，民国传入中国，1955 年后普及
- 原理：石膏吸水不变形，泥浆注入石膏模，石膏吸干水分后泥附着成形
- 可批量复制，厚薄均匀，节省人力
- **辨别注浆坯（识破仿古瓷的关键）**：
  1. 厚薄均匀——手工坯有蓄泥造成的厚薄变化
  2. 圈足直且矮——手工坯圈足高且有斜度
  3. 重量轻——注浆坯更薄
  4. **哈夫线**（half line）：模具拼接处的印迹，可抹平但无法消除

## 宋代成形技术特点
- 拉坯为圆形器主流（轮车技术在宋代已成熟）
- 印坯（外模）用于复杂器形和批量复制（定窑印花即基于此）
- 利坯技术使得薄胎瓷成为可能
- 方形器以雕镶法成形（宋代已有镶器）
- 尚未出现石膏注浆——这是民国以后的技术分水岭
