# 多元釉

## 概述
将多种原料混合制釉，以追求丰富的色彩变化。唐代以前石灰釉只有青色一种玩法，多元釉打开了颜色釉的世界。

## 从单一到多元

### 唐代的突破
- 鲁山窑和禹州窑瓷工往釉中掺入黏土→金属元素丰富→色彩变幻
- 鲁山花瓷：黑釉白斑——黑色来自黏土中的铁，白斑来自草木灰中的铝
- 虽然粗糙，却为宋代的窑变釉打下基础

### 宋代的精化
大窑口放弃成分不稳定的黏土，改用：
- **本釉**：成分简单的瓷土做底釉（氧化硅+氧化铝）
- **矿物着色剂**：替代黏土中的金属
- **草木灰**：担当助熔剂

宋代经典多元釉配方：

| 窑口 | 配方 |
|------|------|
| 磁州窑/景德镇白釉 | 本釉 + 釉灰 |
| 龙泉窑/临汝窑青釉 | 本釉 + 釉灰 + **紫金土**（铁着色） |
| 钧窑红釉 | 本釉 + 釉灰 + **孔雀石**（铜着色）+ 牛骨粉（磷乳浊） |

### 后来的发展
- 元代：青花钴蓝
- 明代：五彩、斗彩
- 清代：粉彩、珐琅彩、唐英 57 种颜色釉

## 核心原理
多种金属元素藏匿于黏土和矿石中，历经火炼呈现变幻莫测的色泽：

| 金属 | 颜色 |
|------|------|
| 铁 | 青、绿、黄、褐、黑 |
| 铜 | 红、紫、绿 |
| 钴 | 蓝 |
| 锰 | 紫 |

> "当年鲁山窑瓷工随手在釉水中掺进一把黏土时，无意间打开了一扇大门，门外的铜红、铁黄、锰紫、钴蓝，扑面而来。"
