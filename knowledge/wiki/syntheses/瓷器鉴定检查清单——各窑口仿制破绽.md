# 瓷器鉴定检查清单——各窑口仿制破绽

## 一、汝窑

### 真品特征
- 胎：香灰胎（灰白），薄而坚致，烧成温度低（1150℃左右），叩之发闷（生烧），重量偏轻不压手 ✅wiki/entities/汝窑
- 釉：天青釉，乳浊失透，气泡「寥若晨星」，开片蟹爪纹/鱼鳞纹/蝉翼纹。玛瑙入釉 → 40倍显微镜下可见珍珠状玛瑙结晶体布于气泡周边 ✅wiki/entities/汝窑 🌐[1][2]
- 形：无大器（不高于30cm），裹足支钉烧，支钉痕细小如芝麻（3/5/6枚），多靠近圈足内墙 ✅wiki/entities/汝窑
- 款：极少有款，仅见「奉华」「蔡」「成王殿」等刻款

### 仿品最容易露馅的环节（按破绽严重度排序）

1. **胎体过厚 + 过重**（致命破绽）
   - 汝窑胎极薄，仿品为防变形把胎做厚，手感压手
   - 真品叩之闷哑（生烧）；仿品叩之声脆（烧结充分）
   - 🌐[1][2]

2. **支钉痕不对**（高价值破绽）
   - 仿品支钉痕或过大或过小，颜色偏白
   - 真品支钉痕为灰白色，极小如芝麻
   - 🌐[2]

3. **釉的质感不对**（最核心破绽）
   - 真品：天青乳浊失透，温润如玉，无玻璃光
   - 仿品：釉色偏蓝偏钧釉，或透明度过高光亮刺目，或开片过密成网状（真品错落有致）
   - 真品气泡「寥若晨星」+ 玛瑙结晶体珍珠状；仿品气泡多如繁星密布
   - 🌐[1][2][3]

4. **做旧痕迹**
   - 酸蚀/土埋 → 釉面呆板 vs 真品自然温润
   - 仿品加乾隆御题诗 → 字体远不如乾隆内府刻工
   - 🌐[2]

## 二、官窑（南宋官窑）

### 真品特征
- 胎：黑胎或深灰胎，紫口铁足（自然还原呈色，非人工涂抹）✅wiki/entities/官窑
- 釉：失透乳浊釉，粉青/天青/月白/油灰/米黄，多次施釉厚达1mm+，釉中气泡「聚沫攒珠」；开片有鳝血纹、金丝铁线；釉面有酥光（丝绸光泽与羊脂玉之间）✅wiki/entities/官窑 🌐[4][5]
- 形：仿青铜礼器（尊鼎炉觚），圈足底窄平、足内墙深外墙浅
- 烧制：先素烧 → 多次施釉 → 还原焰烧成，薄胎厚釉为主

### 仿品最容易露馅的环节

1. **「紫口铁足」造假**（最容易识破）
   - 真品：胎中高铁自然还原呈灰紫/黑褐，过渡自然
   - 仿品：白胎涂酱色釉/化学染色冒充 → 颜色生硬，易脱落
   - 🌐[6]

2. **釉光不对**（最核心）
   - 真品：酥光（介于丝绸与羊脂之间），精光内蕴，绝不刺眼
   - 仿品：或玻璃光贼亮，或酸蚀后乌光呆滞——两者都无法还原「酥光」
   - 🌐[5]

3. **气泡做不到「聚沫攒珠」**
   - 真品：30倍放大镜下气泡平齐紧密，80倍下深浅层次分明，呈「立体感」
   - 仿品：气泡要么均匀细小，要么全无，无法做出大小套叠的聚沫攒珠
   - 🌐[5][6]

4. **开片人工染色**
   - 真品鳝血纹是千年来自然渗出（胎内铁元素缓慢氧化渗入裂纹）
   - 仿品化学染色 → 颜色浮于表面，可擦掉
   - 🌐[5]

5. **器型比例失调**
   - 真品线条端庄典雅，仿品生硬拘谨，颈肩转折不自然
   - 🌐[6]

## 三、哥窑

### 真品特征
- 胎：黑胎（紫金土），紫口铁足自然形成 ✅wiki/entities/哥窑
- 釉：失透乳浊釉，粉青/月白/灰青/青黄。核心特征：
  - 金丝铁线：铁线（黑色大纹，出窑后人工染色）+ 金丝（赭黄小纹，年代氧化形成）
  - 攒珠聚球：气泡大小两种，较小者密如攒珠，较大者呈圈排列如聚球
  - 开片密而不疏，曲而不直
- 形：炉/瓶/洗/盘/碗，传世不足百件
- ⚠️ 窑址未发现，身世存疑 ✅wiki/entities/哥窑

### 仿品最容易露馅的环节

1. **「攒珠聚球」做不到**——这是历代仿品（含雍正乾隆高仿）均无法复制的终极破绽
   - 真品在放大镜下可见大小两类气泡规律排列
   - 仿品气泡要么全无，要么均匀细密如鱼子
   - 🌐[7][8]

2. **金丝铁线是化学做出来的**（最大破绽）
   - 真品：铁线深入胎骨，触摸微涩；金丝是年代氧化自然形成，浮于釉表，迎光温润
   - 仿品：化学药水强蚀→线条生硬如刀刻，粗细均匀；金线泛贼亮荧光；铁线浮在釉表一擦即褪
   - 真品铁线与金丝交错如蛛网不规则；仿品平行或网格状规整
   - 🌐[7][9]

3. **紫口铁足造假**
   - 真品紫口是胎骨含铁自然高温还原呈灰紫，铁足是露胎氧化呈深褐至黑褐，过渡自然
   - 仿品紫口艳如胭脂，铁足乌黑发亮「像刷了鞋油」
   - 🌐[7]

4. **胎体不对**
   - 真品：紫金土胎，断面香灰或浅褐色，叩之声闷
   - 仿品（明清景德镇）：白胎，口沿涂乌金釉模仿紫口 ✅wiki/entities/哥窑 🌐[8]
   - 现代仿品：胎体过于均匀致密，断面洁白或泛青

5. **器型款识破绽**
   - 真品传世哥窑无任何带宋代年款者
   - 凡「成化年制」「雍正年制」哥窑款者 → 明清仿品（非宋）
   - 🌐[7]

## 四、定窑

### 真品特征
- 胎：白胎极细密（似淀粉），薄而轻，一元配方 ✅wiki/entities/定窑
- 釉：象牙白（白中微闪黄），施釉薄，半透明
  - 三大标志：竹丝刷痕、蜡泪痕、玉质感
  - 折腰处积釉呈浅黄绿色
- 形：覆烧 → 芒口（口沿无釉露胎），高档品镶金银铜扣「金装定器」
- 纹：刻花/划花/印花精美

### 仿品最容易露馅的环节

1. **釉色做不出「象牙白」**（最核心）
   - 真品：白中微闪黄，温润如象牙/白玉
   - 仿品：不是偏白就是偏黄；折腰处不见浅黄绿色积釉
   - 🌐[10][11]

2. **竹丝刷痕和蜡泪痕缺失**
   - 真品胎面有竹丝刷修胎的细密旋纹（透过薄釉可见）；盘碗外壁有蜡泪状垂流釉
   - 仿品常无此二特征，或做得生硬拙劣
   - 🌐[11][12]

3. **芒口做假**
   - 真品：覆烧工艺自然形成的芒口，窄而规整
   - 仿品：硬器敲毛口沿 → 涂污垢做旧 → 冒充芒口。仔细观察可见人工敲击痕
   - 🌐[11]

4. **胎质差异**
   - 真品：胎骨洁白极细腻如淀粉，结构紧密无缝隙
   - 南定（景德镇仿）：釉色如粉，胎较白，烧结温度高 → 玻璃质强
   - 土定：胎土粗，偏厚，釉偏黄，釉面有纹片
   - 🌐[11]

5. **红定/紫定罕见**
   - 市场见「红定」须特别小心——至今未见完整实物，仅残片见酱釉上有红斑
   - 紫定实际是酱黑色釉，非紫色
   - 🌐[11]

## 五、钧窑

### 真品特征
- 胎：厚胎，灰色致密，手拉坯 → 底厚口薄 ✅wiki/entities/钧窑
- 釉：厚釉，天青/天蓝/月白底 + 铜红窑变斑。蚯蚓走泥纹（釉层流动的曲线隐纹）；棕眼（釉面细密小坑）
- 形：仿铜器（尊瓶洗觚）+ 花盆，底刻数目字（「一」至「十」）
- 烧制：先素烧 → 施2-3次釉 → 还原焰烧成 ✅wiki/entities/钧窑

### 仿品最容易露馅的环节

1. **蚯蚓走泥纹缺失或伪造**（标志性破绽）
   - 真品：釉层中弯曲不规则的流动状隐纹，仅在较大器物上出现，是素烧后施厚釉的自然结果
   - 仿品：绝大多数无此纹。少数高仿虽做出，但与自然形成有差异
   - 雍正仿钧：红釉色淡、釉薄、白胎，不见蚯蚓走泥纹
   - 🌐[13][14]

2. **窑变斑色彩不对**
   - 真品：紫红色窑变斑自然过渡无固定边界，各色相互熔合流淌
   - 仿品：窑变斑有固定边界（人工洒釉），偏浅或过深，过渡生硬
   - 🌐[13][14]

3. **成型工艺暴露**
   - 真品：手拉坯 → 内壁有轮指痕，圈足露釉处见细密旋纹，底厚口薄
   - 仿品：注浆成型 → 胎壁厚薄均匀，分量轻；两侧有模具接缝痕；有的在内壁故意做假轮指痕但粗糙不均
   - 🌐[13]

4. **做旧方法识别**
   - 兽皮摩擦去浮光 → 放大镜见无数平行摩擦痕
   - 酸蚀浸泡 → 釉面呆板，放大镜见伤痕
   - 油布抛打 → 触摸有油质感
   - 粘合剂做土锈 → 遇水即化
   - 人为剥釉（底足）→ 剥釉断面为斜茬（真品为直茬）
   - 🌐[13]

5. **胎色与烧结度**
   - 真品胎色深灰，淘练精细，致密
   - 仿品胎色偏浅，断面有杂质或孔隙
   - 现代仿品胎细白，火候高，叩之声脆
   - 🌐[14]

## 六、龙泉窑

### 真品特征
- 胎：白胎（弟窑）或黑胎（哥窑系），南宋胎质坚硬细腻，玻化良好 ✅wiki/entities/龙泉窑
- 釉：石灰碱釉（高温粘度大，可施厚釉），南宋巅峰：粉青（乳浊偏蓝）和梅子青（晶莹偏绿）
  - 放大镜下见大量均匀密布的微小气泡
- 形：南宋素面（因厚釉遮挡纹饰），器形古朴稳重；铁足/朱砂足不规则

### 仿品最容易露馅的环节

1. **釉色缺乏内敛厚实感**（最核心）
   - 真品粉青/梅子青莹润如玉，精光内蕴
   - 仿品即使用传统配方（草木灰入釉），质感虽像但浮光隐现，缺乏厚实内敛
   - 酸蚀去光处理 → 釉面损伤在高倍显微镜下暴露
   - 🌐[15][16]

2. **器型线条呆板**
   - 真品颈肩部转折自然流畅
   - 仿品线条生硬，颈肩转折十分生硬，足端部过于整齐，比例失调
   - 🌐[15]

3. **装烧痕迹不对**
   - 南宋铁足/朱砂足往往不规则，是垫饼/垫圈自然痕迹
   - 仿品足部处理过于规整
   - 真品早期产品圈足部分多少有些残疵
   - 🌐[15]

4. **刻划纹饰比较**
   - 真品线条婉转流畅，生动自然
   - 仿品刻划呆板生硬
   - 🌐[15]

5. **元代露胎装饰是识别点**
   - 元代龙泉特有的葱绿釉 + 露胎赭红，仿品难以还原此配色关系
   - 🌐[15]

## 七、建窑（建盏）

### 真品特征
- 胎：铁胎（紫褐色），粗而坚硬，重如铁渣，胎土含沙颗粒感明显 ✅wiki/entities/建窑
- 釉：黑釉厚至垂流成滴珠状，外壁只上半截釉
  - 兔毫：釉中铁结晶拉丝（1300-1350℃）
  - 油滴：铁结晶圆形斑点（1260-1300℃）
  - 口沿釉薄呈黄褐色（氧化铁），有自然侵蚀的麻子底
- 形：敛口/敞口，口大足小如漏斗，修胎草率见棱角痕，底足浅挖近似实足

### 仿品最容易露馅的环节

1. **胎体过轻过细**（致命破绽）
   - 真品：铁胎粗重，胎厚0.2-0.8cm，底部超1cm，压手感强烈
   - 仿品：胎土淘洗过细，胎体表面光滑，较薄，手感轻飘
   - 🌐[17][18]

2. **釉面浮光亮眼**
   - 真品：釉面柔润深沉
   - 仿品：浮光亮眼扎眼（酸蚀则呆板晦涩）
   - 🌐[17]

3. **兔毫丝形态不对**
   - 真品：兔毫丝（铁晶体聚集物）凹陷于黑釉面之下，抚摸有凹凸感，45度光照可见
   - 仿品：兔毫丝与黑釉在同一平面上，釉面平滑光亮
   - 真品口沿自然侵蚀有坑洼麻子底；仿品用金刚砂轮打磨伪造 → 痕迹不同
   - 🌐[18]

4. **垂流釉的差异**
   - 真品外壁垂流釉如烛泪般厚而不规则凝结，参差不齐自然
   - 仿品垂流釉过薄、过整齐，距圈足较近，人工做作
   - 🌐[18]

5. **「供御」刻款**
   - 真品「供御」刻款字迹潇洒飘逸
   - 仿品字迹甚差
   - 🌐[17]

6. **曜变天目**
   - 存世仅3件全在日本。凡市场出现「曜变」→ 必假
   - 🌐[17]

## 八、吉州窑

### 真品特征
- 胎：米黄色（或灰白），含沙量大→粗松吸水强，看似陶但敲之金属声 ✅wiki/entities/吉州窑
- 釉：黑釉为主，独门绝技：
  - 木叶盏：真树叶贴坯→烧飞→留金色叶脉，边缘自然卷曲
  - 剪纸贴花盏
  - 玳瑁盏：黑黄两釉渗化交融
- 形：碗盏多芒口/敛口/深腹，底足矮内凹

### 仿品最容易露馅的环节

1. **胎质做不出「粗砂金属声」**
   - 真品：胎含沙量大自然粗松，吸水强似陶，但敲击有金属声
   - 仿品：高岭土胎质光滑，色泽浅，不含沙。早期仿品掺水泥浆 → 胎深灰但硬度不够落粉；中期掺石膏粉 → 低温烧成叩之声闷
   - 🌐[19][20]

2. **木叶盏的叶纹差异**（最核心）
   - 真品：树叶边缘自然卷曲，叶脉清晰有碳化痕迹，似火舔过；部分叶片似融入碗体
   - 仿品：线条生硬如印刷；边缘整齐无卷曲
   - 真品木叶盏存世极少（比元青花还少），全球仅几十件
   - 🌐[20][21]

3. **做旧手法**
   - 涂胶过火仿土锈 → 沾水即化，手指沾唾沫涂于锈面有土色
   - 老器底敷新胎（接底法）→ 上下敲击声音异样
   - 粗糙老坯重新上釉窑变 → 观察胎釉结合部（老品胎釉合一自然，新品有掉釉起皮）
   - 🌐[19]

4. **施釉方式**
   - 真品：北宋之前半截釉，南宋前釉不见底，晚唐几乎满釉仅露足
   - 🌐[19]

5. **木叶盏器型**
   - 凡过大尺寸（梅瓶类）带木叶纹 → 高度存疑
   - 🌐[21]

## 九、耀州窑

### 真品特征
- 胎：灰褐或灰白色，胎泥淘练不太精细，瓷化程度不高，较薄 ✅wiki/entities/耀州窑
- 釉：青中微黄（橄榄青），釉层莹润透明，绝无流釉。放大镜下明显鱼子纹气泡。底胎釉交接处呈黄褐色或酱色小斑块
- 纹：刻花刀法犀利流畅，主纹深辅纹浅，有层次感似浅浮雕——「古今刀法第一」
- 足：圈足切削平齐规整，足墙细薄

### 仿品最容易露馅的环节

1. **釉色缺黄**（鉴定关键）
   - 真品：不论深浅都含黄的成分——年代越晚闪黄越重。这是耀州窑鉴定的重要依据
   - 仿品：釉色不对，缺少黄调
   - 🌐[22]

2. **刻花刀法差距**（最大破绽）
   - 真品：刀锋圆活犀利有力，主纹深辅纹浅有一定倾斜度，层次感强如浅浮雕，线条行云流水
   - 仿品：从真品倒模 → 层次图案不清晰，浅薄无漂浮感；刻花杂乱无章法，线条绵软无力
   - 🌐[23][24]

3. **底足特征不对**
   - 真品：圈足切削平齐规整，足墙细薄；底胎釉交接处有黄褐色或酱色小斑块（胎中铁高温氧化所致）
   - 仿品：圈足圆滑（与真品切削平齐明显不同），无酱色小斑块，胎色灰白偏浅
   - 🌐[22][23]

4. **胎釉交接的酱色小斑块**
   - 真品漏釉露胎处呈现酱色氧化铁小斑块→这是胎中含铁高的标志，仿品无法仿出
   - 🌐[23]

5. **气泡特征**
   - 真品放大镜下可见明显「鱼子纹气泡」
   - 仿品釉薄不润，气泡稀疏甚至没有
   - 🌐[22]

## 十、磁州窑

### 真品特征
- 胎：大青土，胎质疏松（釉熟胎不熟），颗粒感强，粗/干/硬 ✅wiki/entities/磁州窑
- 釉：透明釉厚润如冰似玉，白化妆土纯度高温润白亮。宋金釉面有细密米子开片
- 纹：白地黑花（斑花石——氧化铁矿物绘制），画法写意流畅一气呵成。珍珠地划花平滑自然
- 形：瓷枕产量最大，手工托坯+竹刀修坯，保留自然刮削痕

### 仿品最容易露馅的环节

1. **注浆胎 = 必假**（铁律）
   - 磁州窑注浆成型技术约在1948年后才采用。凡浆胎者为后仿无疑
   - 🌐[25][26]

2. **胎质不对——釉熟胎不熟的矛盾**
   - 真品：胎质疏松不致密 + 釉面融化很好（古代长时间柴烧、温度略低）
   - 仿品：过度烧结 → 釉薄而胎过度收缩，比重大手感沉重（现代气窑快烧）
   - 🌐[25][26]

3. **白地黑花绘画水平差距**（最容易判断）
   - 真品：用笔流畅一气呵成，潇洒瑰丽，婴戏人物生动有神韵
   - 仿品：拘禁死板，人物有眼无珠，手部与表情表现拙劣。即使用心临摹也缺写意神韵
   - 🌐[26][27]

4. **黑彩用料不同**
   - 真品：斑花石（氧化铁），釉下彩有厚料（有凸感）和薄料之分，铁的还原呈色显褐色或铁锈色
   - 仿品：购买成品料，薄彩居多（轴上无凸感），色调单一
   - 🌐[25]

5. **珍珠地造假**
   - 真品：表面平滑，色朱红/暗红/黑褐
   - 仿品：表面凹凸不平，手绘珠地大小不一
   - 🌐[25]

6. **支钉痕和粘疤**
   - 真品多有古代叠烧遗留的支钉痕
   - 仿品（现代电/气窑平烧）无支钉痕
   - 🌐[25]

## 十一、青花瓷（元青花 + 明青花）

### 真品特征
- 胎：元青花二元配方（高岭土+瓷石），胎质不如明清细致，砂底有砂眼/刷痕/铁质斑点，底足露胎处见火石红 ✅wiki/entities/青花瓷
- 釉：元青花白中泛青，积釉处呈湖蓝色（非湖绿）；明青花釉面肥厚滋润
- 青料：进口苏麻离青 → 靛青泛紫，浓艳，有黑褐色铁锈斑，斑上呈「锡光」；国产料 → 灰蓝
- 纹：元青花层次多（可达7-8层），绘画气韵连贯，笔触感强。云肩纹、莲池鱼藻、缠枝牡丹、人物故事为经典

### 仿品最容易露馅的环节

1. **苏麻离青铁锈斑造假**（最核心破绽）
   - ①点氧化铁法（最拙劣）：烧成后点氧化铁复烧→细小点子，可用指甲刮下
   - ②钻眼点铁法：先钻小眼再点氧化铁→沾水一抹+指甲刮→立判
   - ③釉下添加金属法（最具欺骗性）：青花料上涂含铁/铝/锡物质→烧后成片斑痕。鉴别：聚光手电+放大镜→可见釉下两层（上层锈斑浮在青花上，下层青花），有立体感
   - 🌐[28]

2. **青花发色不对**
   - 真品元青花：靛青中泛紫或青翠，浓淡不一，有自然铁锈斑
   - 仿品：发灰或色调单一，浓淡无规律；浓处是「有意添笔加深」
   - 仿品积釉处呈湖绿色（应为湖蓝色）
   - 🌐[29][30]

3. **绘画水平差距**
   - 真品：运笔畅通，起/行/收笔自如，笔触感强，气韵连贯，龙纹凶猛有力
   - 仿品：用笔无力生硬，缺乏层次，画面拘谨粗陋，龙纹无气势
   - 真品鱼藻纹「不画水波却使人感到鱼翔浅底」
   - 🌐[30]

4. **底足处理暴露**
   - 真品大器：砂底，底足欠规整有弯曲状带削痕，火石红自然
   - 仿品：故意加「火石红」（薄薄一层极不自然），底足处理生硬
   - 真品罐类口大足小（元）vs 仿品口足比例不对
   - 🌐[30][31]

5. **款识破绽**
   - 真品元青花：极少带款，至正型无年款
   - 仿品：画蛇添足加「大元成吉思汗纪国器上内府公用至正元年九月」→ 一眼假
   - 🌐[30]

6. **胎质差异**
   - 真品元青花：硬中见柔，胎质淘练不如明清细致
   - 仿品：胎质过细过白或过粗过硬
   - 🌐[31]

## 通用做旧手法识别（跨窑口适用）

| 做旧手法 | 效果 | 破绽 |
|---------|------|------|
| 酸蚀/强碱浸泡 | 去浮光，仿出土 | 釉面呆板，放大镜见腐蚀伤痕 |
| 兽皮/砂纸摩擦 | 去浮光 | 放大镜见无数平行摩擦痕 |
| 油布抛打 | 仿柔润光泽 | 触摸有油质感 |
| 土埋/粘合剂做土锈 | 仿出土 | 粘合剂遇水即化，手指沾唾沫可擦掉 |
| 人为剥釉（底足） | 仿年代久远 | 真品剥釉断面直茬，仿品斜茬 |
| 老胎新釉/接底 | 半真半假 | 上下敲击声音异样；胎釉结合部掉釉起皮 |
| 涂胶过火 | 仿土锈 | 沾水脱落 |

## 游戏玩法建议

基于以上清单，可设计以下鉴定机制：

1. **五维评分制**：胎/釉/形/纹/款各占一定权重，每个维度根据破绽数量给出「可疑指数」
2. **关键破绽一击必杀**：某些窑口的独有特征如缺失 = 直接判定为仿品（如哥窑无攒珠聚球、建盏胎太轻、汝窑支钉痕过大）
3. **窑口盲猜模式**：只展示胎/釉/形/纹/款的片段信息，玩家需要推断窑口
4. **做旧手法识别**：额外维度——识别器物表面的做旧痕迹
