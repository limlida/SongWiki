# 青花瓷的全球循环

## 核心论述

青花瓷经历了完整的全球化循环——技术来自西亚、产品卖向西亚和欧洲、被日本和欧洲模仿、反过来影响中国。

### 第一循环：波斯钴 → 中国青花 → 穆斯林市场

```
波斯钴料（苏麻离青）
    ↓
唐代：巩县窑尝试钴蓝彩绘
    ↓
元代：磁州窑+吉州窑工匠迁景德镇 + 波斯人带来钴料 + 景德镇白瓷
    → 青花瓷一蹴而就
    ↓
穆斯林大批采购（土耳其托普卡帕宫藏40+件元青花）
```

### 第二循环：克拉克瓷 → 欧洲疯狂 → 日本模仿 → 中国反仿

```
1603：荷兰海盗劫获葡萄牙商船→克拉克瓷阿姆斯特丹拍卖→轰动欧洲
    ↓
欧洲"瓷器病"→大量采购→白银单向流入中国→通货膨胀
    ↓
中国战乱+海禁→日本有田烧填补市场（伊万里烧）
    ↓
康熙中期：中国恢复元气→欧洲人要求景德镇生产"伊万里风格"瓷器
    → 中国人仿日本仿中国的青花瓷——完整的循环
```

### 第三循环：殷弘绪 → 欧洲自产瓷器 → 工业化碾压

```
1712：殷弘绪信寄到欧洲→公布全部72道工序
    ↓
（梅森瓷1710年已先于殷弘绪信烧成硬质瓷）
    ↓
欧洲瓷厂遍地开花→蒸汽机+贴花+注浆→工业化生产
    ↓
晚清：中国从瓷器出口国变为进口国
```

### 循环的本质

青花瓷从来不是"纯中国"的——它的蓝色来自波斯、器形迎合穆斯林、画面适应欧洲。**封闭的系统是反动的**（有泉语）。陶瓷史就是一部中西文化交流融合的历史。
