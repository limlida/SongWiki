# 技术审美与文人审美的张力

## 核心论述

陶瓷史上存在两条审美路线：**文人审美**追求冲淡典雅（汝窑、雍正官窑），**技术审美**追求极致工艺（乾隆官窑）。二者并非谁对谁错，而是互相成就。

### 文人审美线

| 代表 | 特征 |
|------|------|
| 汝窑 | 天青釉色、素面无纹、裹足支钉烧——"雨过天青云破处" |
| 官窑 | 开片为美、酥油光、仿青铜器形——徽宗的克己复礼 |
| 雍正官窑 | 清雅含蓄、国画入瓷、精确到"叶子要什么颜色" |
| 龙泉窑 | 粉青梅子青、放弃刻花纯以釉色悦人 |

### 技术审美线

| 代表 | 特征 |
|------|------|
| 钧窑 | 铜红釉突破、窑变花釉——"钧瓷无双" |
| 乾隆官窑 | 各种釉彩大瓶、转心瓶、仿生瓷——"把技术发挥到极限，穷尽所有可能" |
| 唐窑 | 57种颜色釉——"基本上想烧什么就烧什么" |

### 有泉的辩证

> "乾隆替后人翻了筋斗"——看过浮华盛极的乾隆瓷，回头再看雍正和汝窑，感受必然不同。而这种感受，恰恰是雍正和徽宗未尝领略过的。

- 雍正的典雅建立在对技术的完全掌控之上（唐窑的功劳）
- 乾隆的浮华是技术审美的阶段性爆发
- "技术审美源自对于人类自身能力的沉迷和赞叹"

### 文人壶：第三条路

紫砂壶由文人主导（陈曼生→顾景舟），在技术和审美之间找到平衡：
- 技术：泥片拍打成形、砑光、调砂
- 审美：诗书画印入壶、朴素内敛、"日用之雅"
- 铭文："汲井匪深，挈瓶匪小。式饮庶几，永以为好"
