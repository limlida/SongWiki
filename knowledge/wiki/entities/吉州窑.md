# 吉州窑

> [孤证警告] 仅有一篇 source 支持。

## 概述
窑址江西永和（又名永和窑），宋朝大窑，"姑苏慕容"风格——仿定窑白瓷、磁州窑白地黑花、龙泉窑青瓷、镇窑青白瓷、官窑开片青瓷，"天下瓷器，尽入吾彀中" ^[陶瓷简史:L2141]。黑釉盏与建窑平分秋色，但用二次施釉取巧 ^[陶瓷简史:L2145]。文天祥是吉州永和人。

## 核心内容

### 黑釉盏：取巧路线
- 建窑：单一黑釉烧出油滴兔毫，难度大
- 吉州窑：二次施釉——先烧黑盏 → 釉面施浅色强流动性釉 → 再低温烧 → 色彩和变化比建盏强烈 ^[陶瓷简史:L2145]
- 鹧鸪斑：先烧黑釉盏 → 用笔或淋画白釉 → 再低温烧。格调远逊建阳油滴但名字好听 ^[陶瓷简史:L2148]
- 黄庭坚《满庭芳》提及鹧鸪斑盏 ^[陶瓷简史:L2148]

### 独创品种
- **玳瑁**：黑黄两釉渗化交融效果，似海龟壳 ^[陶瓷简史:L2154]
- **木叶盏**：真树叶粘坯→上釉入窑→树叶烧飞→留下金色叶脉。斗笠形，桑叶居多，现代高手用酸腐蚀叶肉+叶脉蘸黄釉仿制可乱真 ^[陶瓷简史:L2156]
- **剪纸盏**：剪纸粘盏烧制，龙凤/花卉/生肖/吉语，颇接地气 ^[陶瓷简史:L2158]

### 衰落
文天祥招兵说+窑变砸窑说 → 但吉州窑元代仍在生产。真正停烧在明代——废团茶→斗茶消亡 → 黑釉盏无市场 ^[陶瓷简史:L2174-L2176]
