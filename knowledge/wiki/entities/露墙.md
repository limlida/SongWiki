# 露墙

## 概述

露墙是"袒露之墙、露天之墙"，不附着于房屋基础上，也不附属于房屋梁柱之旁。典型用途包括院落之间的院墙，或建筑群四周的围墙。^[营造法式建筑高度考:L205]

## 核心内容

### 高厚比与收分

据《营造法式》卷第三·壕寨制度·墙：^[营造法式建筑高度考:L209-215]

| 规则 | 数值 |
|------|------|
| 基准高厚比 | 墙高1丈 → 基厚0.5丈（基厚=高×½） |
| 收分 | 至墙顶，厚约0.2丈（"上收面之广，比高五分之一"） |
| 高度增量 | 每增1尺 → 厚+3寸；每减1尺 → 厚-3寸 |

### 「上收面之广」释义

此句含意不太明确，有两种解释：
1. 指两面斜收之广共为高的五分之一
2. 指墙顶所余净厚度为高的五分之一，即高1丈→顶厚2尺（64cm）^[营造法式建筑高度考:L213]

### 城墙关联

若露墙中使用了草葽、木橛等加固材料，功计法与夯筑城墙时一致。^[营造法式建筑高度考:L215]
