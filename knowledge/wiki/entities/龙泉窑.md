# 龙泉窑

> [孤证警告] 仅有一篇 source 支持。

## 概述
位于浙江龙泉，北宋越窑工匠南迁建立。青瓷集大成者——南宋把青釉发展到极致，全面超越越窑 ^[陶瓷简史:L2013]。发明石灰碱釉（降低CaO+加入K₂O→黏度高→可厚施十几层→釉比胎厚），烧出粉青和梅子青两种传奇釉色 ^[陶瓷简史:L2017]。一直是民窑，贡窑都没做过，"怀才不遇" ^[陶瓷简史:L2013]。

## 核心内容

### 起源与创始人
- 龙泉窑兴起源自越窑衰落 ^[陶瓷简史:L1983]
- 北宋早期越窑工匠南迁至龙泉 → "农忙而耕，农闲而陶"的田园生活 → 越窑专业团队加入 → 史上最伟大青瓷产地 ^[陶瓷简史:L1985-L1988]
- 章生一/章生二兄弟：明代起被奉为祖师 ^[陶瓷简史:L1994]。琉田窑（龙泉西南40公里，主窑址至明，近百座古窑炉）^[陶瓷简史:L1998]
- 龙泉哥窑（开片铁胎，占窑址1/10）vs 弟窑（不开片白胎） ^[陶瓷简史:L2002]

### 技术巅峰：石灰碱釉
- 降低氧化钙比例+加入氧化钾 → 黏度高不流淌 → 可反复施釉十几层 ^[陶瓷简史:L2017]
- 粉青：青中偏蓝、淡雅柔和、有乳浊效果 ^[陶瓷简史:L2017]
- 梅子青：比粉青厚、温度更高、晶莹绿色如青梅 ^[陶瓷简史:L2017]
- 放弃刻花 → 厚釉遮掩 → 素面纯以釉色悦人 ^[陶瓷简史:L2017]

### 各朝演变
- 北宋：日用器为主，石灰釉薄偏褐黄，划花/刻花/贴花 ^[陶瓷简史:L2005]
- 南宋：巅峰期，受官窑影响烧礼器（香炉/琮式瓶）+文房+鸟食罐 ^[陶瓷简史:L2007]
- 元代：釉变薄、颜色不及南宋、器形大（迎合外销）；露胎装饰+褐斑 ^[陶瓷简史:L2019-L2021]
- 明代：衰落——镇窑永乐仿龙泉釉+官窑在景德镇→失去官方订单→产品线单一 ^[陶瓷简史:L2041]；银矿禁采+叶宗留起义（1445-1451）→错失出口转型 ^[陶瓷简史:L2043-L2048]
- 1957年周恩来《关于恢复历史名窑的决定》→恢复 ^[陶瓷简史:L2052]

### Celadon（雪拉同）命名
三种说法：法国歌剧牧羊人Celadon（时间对不上）；梵语（中国瓷未到印度）；Saladin（萨拉丁）——1171年送苏丹40件龙泉青瓷，最可信 ^[陶瓷简史:L2025-L2029]
