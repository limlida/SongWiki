# 北宋河北边防建设研究（全书结构）

## 概述
王轶英，2007年河北大学硕士论文（导师郭东旭）。80页。系统研究北宋河北边防建设，是目前已知最完整的河北边防专论。^[北宋河北边防建设研究:L1-L50]

|----|------|---------|
| 绪论 | 学术回顾与研究空间 | 前人研究偏重战争和塘泊，缺镇寨建设、边防法规系统研究 ^[北宋河北边防建设研究:L115-L135] |
| 一 | 河北边防战略思想变化 | 收复幽云→雍熙北伐失利后转攻为守→澶渊之盟定型"银帛换和平" ^[北宋河北边防建设研究:L137-L200] |
| 二 | 军事防御战略部署 | 四路安抚使司、驻军布局（常年20-30万）、乡兵体系 ^[北宋河北边防建设研究:L200-L310] |
| 三 | 御边战术运用 | 以步制骑、平戎万全阵、弓弩战术、东西拐子马 ^[北宋河北边防建设研究:L311-L360] |
| 四 | 沿边镇寨建设 | 州军调整、城镇防御设施（瓮城/护城井/料敌塔）、寨铺体系、修城受誓约束 ^[北宋河北边防建设研究:L360-L440] |
| 五 | 沿边防御工程建设 | 塘泊防线（"水长城"）、军事防御林、"地下长城" ^[北宋河北边防建设研究:L441-L530] |
| 六 | 边防安全措施 | 保密制度、防奸细、禁越度、沿边捕盗法 ^[北宋河北边防建设研究:L530-L600] |
