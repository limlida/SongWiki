# 军事防御林

> [孤证警告] 仅有一篇 source 支持。

## 概述
北宋在河北沿边大规模种植榆柳，形成以密林阻隔辽骑的人工屏障。是塘泊防线以西（保州至西山无塘水处）的关键替代防御设施。^[北宋河北边防建设研究:L509-L525]

## 建设历程
| 时期 | 举措 |
|------|------|
| 太祖建隆二年(961) | 诏令"课民植树" ^[北宋河北边防建设研究:L509] |
| 太祖朝 | "于瓦桥一带南北分界之所专植榆柳，中通一径，仅能容一骑" → 至真宗朝"日益繁茂，合抱之木交络翳塞" ^[北宋河北边防建设研究:L510] |
| 仁宗明道二年(1033) | 诏"种木于西山之麓，以法'榆塞'" ^[北宋河北边防建设研究:L509] |
| 仁宗皇祐元年(1049) | 命"自保州以西无塘水处，广植林木，异时以限敌马" ^[北宋河北边防建设研究:L510] |
| 神宗熙宁时 | 沈括奏报"定州北境先种榆柳以为寨，榆柳植者以亿计" ^[北宋河北边防建设研究:L510] |

## 法律保护
- 划为"禁山"，不许斩伐
- 定期检校林木数量，画有"北面榆柳图"，所成数"逾三百万" ^[北宋河北边防建设研究:L512]
- 天禧二年曾中辽反间计砍伐保州等处榆柳，后恢复禁采伐之令 ^[北宋河北边防建设研究:L513]

## 与陶瓷的隐含关联
防御林大面积占用河北土地并限制采伐 → 民用燃料来源进一步收窄 → 间接加剧定窑从柴烧转向煤烧的压力。此关联为推断，论文未论及。
