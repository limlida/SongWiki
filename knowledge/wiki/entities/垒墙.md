# 垒墙

## 概述

垒墙是结合房屋本体用土坯（未经烧制的"墼"）垒砌的围护墙。^[营造法式建筑高度考:L181]

## 核心内容

### 高厚比与收分

据《营造法式·泥作制度》：^[营造法式建筑高度考:L185-189]

| 规则 | 数值 |
|------|------|
| 基准高厚比 | 墙高4尺 → 基厚1尺 |
| 收分 | 每高1尺 → 两侧各斜收3分（共6分） |
| 高度增量 | 每增1尺 → 基厚+2.5寸 |

### 高度

垒墙高度约4尺（128cm）+ 增量1尺（32cm）。^[营造法式建筑高度考:L199]

### 工艺

每三层土坯铺一层"襻竹"——用竹子将土坯墙扣扭在一起，相当于现代墙体中的横向钢筋，增加墙体整体性。^[营造法式建筑高度考:L193]

土坯（"条墼"）标准尺寸：长1.2尺（38cm）× 宽0.6尺（19cm）× 厚0.2尺（6cm）。^[营造法式建筑高度考:L195]
