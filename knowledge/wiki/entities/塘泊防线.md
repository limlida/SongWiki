# entities/塘泊防线

- **别称**：水长城、塘泊屯田防线
- **位置**：北宋河北路沿边，自边吴淀至泥姑海口，绵亘七州军，屈曲九百里
- **来源**：[[sources/北宋河北边防建设研究]]，核心史料《宋史·河渠志》《续资治通鉴长编》
- **置信度**：high
- **最后验证**：2026-06-04

## 功能

"深不可以舟行，浅不可以徒涉，虽有劲兵，不能度也。"以人工蓄水改变平原开阔地貌，限制辽骑兵奔冲速度与路径，为宋军调兵争取时间。

## 管理机构

河北屯田司 + 缘边安抚司共同掌管，河北转运使兼任都大制置。屯田司置屯田郎中、员外郎各一人，"凡塘泊以时增减，堤堰以时修葺"。每季度向工部申报塘水深度。

## 关键人物

- 杨怀敏（领屯田司，大广塘水，曾对仁宗说"臣于水边为陛下置虎翼军一百万矣"）
- 葛怀敏（知雄州时"立木为水则，以限盈缩"）

## 分布（自东向西）

破船淀、灰淀、方淀 → 鹅巢淀、燕丹淀 → 水汶淀、得胜淀 → 粮料淀 → 百世淀、黑羊淀、小莲花淀 → 大莲花淀、白羊淀 → 齐女淀、边吴淀 → 沈苑泊 → 西塘泊

## 澶渊之盟的影响

盟书虽未明文禁增广塘泊，但北宋"自今以后各不添展"的约定使之受约束。边臣言利害虽听许，"必戒之以勿张皇，使契丹有词"。北宋末塘泊堙废，失去防御功能。
